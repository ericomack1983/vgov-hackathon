## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding between icon and label |
| sm | 8px | Compact element spacing, sidebar item internal padding |
| md | 16px | Default element spacing, card padding, input padding |
| lg | 24px | Section padding, sidebar group spacing |
| xl | 32px | Layout gaps between sidebar and content, header height padding |
| 2xl | 48px | Major section breaks, page top/bottom margins |
| 3xl | 64px | Sidebar width gutter (sidebar is 256px / 16rem wide) |

Exceptions: Sidebar width is 256px (16rem). Header height is 64px. Minimum touch target for role switcher dropdown items is 40px height.

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 500 (medium) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 700 (bold) | 1.2 |

Font family: `Inter` loaded via `next/font/google` with `subsets: ['latin']`. Fallback stack: `Inter, system-ui, -apple-system, sans-serif`.

Label is used for sidebar nav items, role switcher labels, and metadata text. Body is used for placeholder page content. Heading is used for page titles. Display is reserved for future dashboard metric values (not used in Phase 1 but declared for consistency).

## Layout Contract

### App Shell Structure

```
+----------------------------------------------------------+
| Header (h-16, bg-slate-900, fixed top, z-30)             |
| [Logo: "GovProcure AI"]     [Role Switcher]  [Bell icon] |
+----------+-----------------------------------------------+
| Sidebar  | Main Content                                   |
| (w-64,   | (ml-64, mt-16, p-8, bg-slate-50,              |
| bg-slate | min-h-screen)                                  |
| -900,    |                                                |
| fixed    | Placeholder page content renders here          |
| left,    |                                                |
| top-16,  |                                                |
| z-20)    |                                                |
|          |                                                |
| Nav      |                                                |
| items    |                                                |
| vary     |                                                |
| by role  |                                                |
+----------+-----------------------------------------------+
```

- Header: fixed, full width, height 64px (`h-16`), `bg-slate-900`, `border-b border-slate-700`
- Sidebar: fixed, left 0, top 64px, width 256px (`w-64`), full height minus header, `bg-slate-900`, `border-r border-slate-800`
- Main content: `ml-64 mt-16 p-8 bg-slate-50 min-h-screen`

### Nav Item States

| State | Style |
|-------|-------|

| Hover | `text-slate-50 bg-slate-800` transition 150ms ease |

Icon size: 20px (`w-5 h-5`). Gap between icon and label: 12px (`gap-3`).

## Role Switcher Contract

- Location: Header, right-aligned before notification bell
- Trigger: Button showing current role name + ChevronDown icon
- Trigger style: `bg-slate-800 text-slate-200 px-3 py-1.5 rounded-lg text-sm font-medium`
- Dropdown: Absolutely positioned below trigger, `bg-slate-800 border border-slate-700 rounded-xl shadow-lg`
- Dropdown items: `px-4 py-2.5 text-sm text-slate-300 hover:bg-slate-700 hover:text-white`
- Selected role: `text-indigo-400 font-medium` with a small indigo dot (6px, `rounded-full bg-indigo-500`) to the left
- Roles: "Gov Officer", "Supplier", "Auditor"
- Animation: Framer Motion `AnimatePresence` with opacity + y-translate (from -4px to 0) over 150ms

## Animation Contract (Phase 1)

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Page mount | Opacity 0 to 1 + translateY 8px to 0 | 300ms | ease-out |
| Sidebar nav item hover | Background color transition | 150ms | ease |
| Role switcher dropdown open | Opacity 0 to 1 + translateY -4px to 0 | 150ms | ease-out |
| Role switcher dropdown close | Opacity 1 to 0 + translateY 0 to -4px | 100ms | ease-in |

All animations use Framer Motion `motion` components. Page transitions use `AnimatePresence` with `mode="wait"`.

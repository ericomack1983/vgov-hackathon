**1. Create `src/lib/audit-utils.ts`:**

```typescript
import { RFP, Transaction } from '@/lib/mock-data/types';

export interface AuditEvent {
  id: string;
  timestamp: string;
  type: 'rfp_created' | 'rfp_published' | 'bid_submitted' | 'evaluation_run'
       | 'supplier_awarded' | 'override_applied' | 'payment_initiated'
       | 'payment_settled';
  description: string;
  rfpId: string;
  rfpTitle: string;
  actor: string;
  metadata?: Record<string, string>;
}
```

Export `function buildAuditTrail(rfps: RFP[], transactions: Transaction[]): AuditEvent[]`:

For each RFP:
- Always add `rfp_created` event with `rfp.createdAt` timestamp, description `RFP "${rfp.title}" created`, actor `Gov Officer`.
- If status is not 'Draft': add `rfp_published` event. Use a timestamp 1 minute after createdAt (mock). Description: `RFP "${rfp.title}" published for bidding`.
- For each bid in `rfp.bids`: add `bid_submitted` event with `bid.submittedAt` timestamp, description `${bid.supplierName} submitted bid of $${bid.amount.toLocaleString()}`, actor `Supplier`, metadata `{ supplierId: bid.supplierId, amount: String(bid.amount) }`.
- If `rfp.evaluationResults`: add `evaluation_run` event. Timestamp = 2 minutes after last bid submitted (or rfp.createdAt if no bids). Description: `AI evaluation completed - ${rfp.evaluationResults.length} bids scored`. Actor: `System`.
- If `rfp.overrideWinnerId`: add `override_applied` event. Timestamp = 1 minute after evaluation. Description: `Manual override applied: ${rfp.overrideJustification}`. Actor: `Gov Officer`.
- If `rfp.selectedWinnerId` and status in ['Awarded', 'Paid']: add `supplier_awarded` event. Use appropriate timestamp. Description: `Supplier awarded for RFP "${rfp.title}"`. Actor: `Gov Officer`.

For each Transaction:
- Add `payment_initiated` event with `tx.createdAt` timestamp, description `${tx.method} payment of $${tx.amount.toLocaleString()} to ${tx.supplierName}`, actor `Gov Officer`, metadata `{ orderId: tx.orderId, txHash: tx.txHash || '' }`.
- If `tx.settledAt`: add `payment_settled` event with `tx.settledAt` timestamp, description `Payment settled - ${tx.method} $${tx.amount.toLocaleString()} to ${tx.supplierName}`, actor `System`.

Sort all events by timestamp descending (newest first). Return the sorted array.

Export a helper `function getEventIcon(type: AuditEvent['type']): string` returning icon names: rfp_created -> 'FileText', rfp_published -> 'Send', bid_submitted -> 'FileInput', evaluation_run -> 'Bot', supplier_awarded -> 'Award', override_applied -> 'AlertTriangle', payment_initiated -> 'CreditCard', payment_settled -> 'CheckCircle'.

Export a helper `function getEventColor(type: AuditEvent['type']): string` returning Tailwind text color classes: rfp_created -> 'text-slate-500', rfp_published -> 'text-indigo-500', bid_submitted -> 'text-blue-500', evaluation_run -> 'text-purple-500', supplier_awarded -> 'text-emerald-500', override_applied -> 'text-amber-500', payment_initiated -> 'text-indigo-500', payment_settled -> 'text-emerald-500'.

**3. Create `src/components/audit/ExportPDFButton.tsx`:**

`'use client'` directive. Import `useState` from react, `Download`, `Loader2` from lucide-react.

Props: `{ reportRef: React.RefObject<HTMLDivElement | null> }`.

State: `isExporting: boolean` (default false).

The `handleExport` async function:
1. Set `isExporting(true)`.
2. Dynamically import both libraries: `const [{ default: jsPDF }, { default: html2canvas }] = await Promise.all([import('jspdf'), import('html2canvas')])`.
3. Get the element from `reportRef.current`. If null, return.
4. Call `html2canvas(element, { scale: 2, useCORS: true, backgroundColor: '#ffffff' })`.
5. Create `new jsPDF('p', 'mm', 'a4')`.
6. Compute dimensions: `pdfWidth = pdf.internal.pageSize.getWidth()`, `pdfHeight = (canvas.height * pdfWidth) / canvas.width`.
7. `pdf.addImage(canvas.toDataURL('image/png'), 'PNG', 0, 0, pdfWidth, pdfHeight)`.
8. `pdf.save('govprocure-audit-report.pdf')`.
9. Set `isExporting(false)`.
10. Wrap in try/catch, on error log to console and set isExporting false.

Render a button: `bg-indigo-600 hover:bg-indigo-700 text-white px-4 py-2 rounded-lg text-sm font-semibold inline-flex items-center gap-2 disabled:opacity-50 transition-colors`. Disabled when `isExporting`. Shows Loader2 (animate-spin) when exporting, Download icon otherwise. Text: "Export PDF" or "Generating...".
